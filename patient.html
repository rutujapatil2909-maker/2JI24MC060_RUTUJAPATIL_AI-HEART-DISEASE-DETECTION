⬡ About the System

Holographic Cardiac Dashboard

CardioViz 3D combines AI video analysis with real-time WebGL rendering to give clinicians and researchers an interactive, anatomically accurate 3D heart model — with disease regions highlighted only where detected.

🫀
3D Heart Visualization
21 anatomical structures rendered as separate procedural meshes with medical color coding. Semi-transparent outer wall reveals inner chambers, valves, and vessels.
WebGL · Three.js
🎬
AI Video Analysis
Upload MP4 angiography or echocardiogram footage. OpenCV extracts frames, enhances contrast, and detects arterial narrowing, blockages, and abnormal flow patterns.
OpenCV · Python
📊
Diagnosis Report
Detailed findings table with severity classification (mild / moderate / severe), affected area percentage, clinical notes, and printable PDF-ready layout.
REST API · Flask
🎛️
Interactive Controls
Transparency sliders per layer, visibility toggles for chambers/vessels/valves, cross-section slicing, blood flow animation, and auto-rotate with manual orbit.
Real-time
🔬
Disease Highlighting
Affected structures glow yellow→orange→red by severity with pulsing animation. All healthy anatomy remains at original medical colors — no false positives.
AI Detection
💾
Export & Share
Export the 3D heart model as OBJ or STL for use in external tools. Capture labeled screenshots with patient info overlay for inclusion in clinical reports.
OBJ · STL · PNG
⬡ System Features
❤️
Anatomically Accurate Model
All 21 structures built from procedural geometry — no placeholder shapes. Correct proportions, organic curves, and realistic vessel branching.
🧠
AI Abnormality Detection
Frame-by-frame analysis detects artery narrowing, blockages, and abnormal flow. Maps findings to specific anatomical regions automatically.
🔭
Transparent Layered View
Individual opacity control per layer. Outer wall at 65% transparency reveals chambers. Vessels and valves independently adjustable.
⚡
Real-time Blood Flow
Animated particle flow through chambers and vessels. Toggle on/off. Follows anatomically correct oxygenated and deoxygenated pathways.
📋
Multi-page Workflow
Patient entry → MP4 upload → 3D dashboard → diagnosis report. Session-based with persistent data across all four pages.
🖥️
Medical-Grade UI
Dark holographic theme with neon accents. HUD panels, ECG monitor, vital signs display, and findings chips — all in a single dashboard view.
